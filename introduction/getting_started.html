Getting Started

Getting Started

이 내용은 KAIST CS330 과정의 Pintos 프로젝트에 대한 소개와 시작 방법에 관한 것입니다. Pintos는 x86-64 아키텍처를 위한 간단한 운영 체제 프레임워크로, 스탠퍼드 대학에서 개발된 프로젝트를 기반으로 합니다. 이 프로젝트는 커널 스레드, 사용자 프로그램의 로딩 및 실행, 파일 시스템을 지원하지만, 모두 매우 단순한 방식으로 구현되어 있습니다. Pintos 프로젝트를 통해 학생들은 이러한 영역들에서의 지원을 강화하고, 가상 메모리 구현을 추가할 것입니다.

주요 포인트

  • 환경 설정: 학생들은 Linux 머신에서 Pintos를 구성하고 실행할 것입니다. 소스 코드는 git clone https://github.com/casys-kaist/pintos-kaist를 통해 얻을 수 있으며, 환경 설정을 위해 source ./activate 명령을 사용합니다. 이는 .bashrc에 추가하는 것이 좋습니다.
  • 소스 트리 구조: Pintos의 디렉토리 구조에는 여러 부분이 포함되어 있습니다.
    • threads/: 프로젝트 1부터 수정할 기본 커널의 소스 코드입니다.
    • vm/: 프로젝트 3에서 가상 메모리를 구현할 거의 비어 있는 디렉토리입니다.
    • filesys/: 기본 파일 시스템의 소스 코드로, 프로젝트 2부터 사용하되 프로젝트 4까지 수정하지 않습니다.
    • devices/: I/O 디바이스 인터페이싱의 소스 코드로, 주로 프로젝트 1에서 타이머 구현을 수정할 것입니다.
  • 빌드 및 실행: make 명령을 사용해 소스 코드를 빌드하고, pintos run alarm-multiple과 같은 명령을 사용해 Pintos를 시뮬레이터에서 실행할 수 있습니다.

이 프로젝트는 학생들에게 운영 체제의 핵심 개념과 설계에 대한 실질적인 경험을 제공하며, 특히 커널 프로그래밍, 사용자 프로그램 관리, 파일 시스템, 가상 메모리 시스템 등의 구현에 중점을 둡니다. 이 과정은 많은 시간과 노력을 요구하지만, 운영 체제에 대한 깊은 이해와 실질적인 경험을 제공할 것입니다.
